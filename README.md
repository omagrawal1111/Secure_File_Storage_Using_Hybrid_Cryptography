# Secure-File-Storage-Using-Hybrid-Cryptography
A flask web app to upload, encrypt and store file using Hybrid Cryptography. 

## Algorithms Used : 
▶ AES Algorithm  <br>
▶ BlowFish Algorithm  <br>
▶ TrippleDES Algorithm  <br>
▶ IDEA  <br>
